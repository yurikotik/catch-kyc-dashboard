# Catch KYC Dashboard

Mobile-optimized stock analysis dashboard using Relative Strength, Momentum, and Z-Score metrics.

## Features

- **Weighted Scoring**: 50% Z-Score, 25% RS, 25% Momentum
- **Mobile Optimized**: Adjustable ticker counts for mobile viewing
- **Interactive Heatmap**: Visual analysis with hover details
- **Full Rankings**: Complete opportunity rankings with download
- **Price Charts**: Multi-ticker comparison and returns analysis

## Live Dashboard

Deployed on Streamlit Cloud for real-time analysis.

## Data

Historical price data for 48 CEF tickers over 125 trading days.

## Usage

1. **Heatmap Tab**: Adjust ticker count for mobile (10-15 recommended)
2. **Opportunities Tab**: View full rankings, download CSV
3. **Metrics Tab**: Analyze individual metrics
4. **Charts Tab**: Select tickers to compare performance

## Weighting

- **Z-Score (50%)**: Fundamental valuation metric
- **Relative Strength (25%)**: Performance vs SPY benchmark
- **Momentum (25%)**: 20-day price trend
